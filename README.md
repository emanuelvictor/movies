# Ambiente

Software requerido:

- Java 11
- Maven

# Testes e acesso

Execute, via linha de comando, as seguintes instruções:

- mvn clean package && java -jar target/movies-0.0.1-SNAPSHOT.jar
- Acesse http://localhost:8080/producers?min=true&max=true
